🔑Login Screen / ការចូលគណនី

Authentication
  • Secure Login System (ប្រព័ន្ធចូលគណនីមានសុវត្ថិភាព): Input validation for email format and empty fields (ការផ្ទៀងផ្ទាត់ទម្រង់អ៊ីមែល និងការពារចន្លោះទទេ)។
  • AuthService.login() (ការផ្ទៀងផ្ទាត់គណនី): Reads local file accounts.json to authenticate (អានឯកសារ accounts.json ក្នុងម៉ាស៊ីនដើម្បីផ្ទៀងផ្ទាត់គណនី)។
  • User Experience (បទពិសោធន៍អ្នកប្រើប្រាស់): Displays clear errors via red SnackBar alerts (បង្ហាញសារព្រមានពណ៌ក្រហមច្បាស់លាស់)។
  • State Injection (ការបញ្ចូលទិន្នន័យ State): Sets active student globally in AppState on success (រក្សាទុកទិន្នន័យសិស្សទៅក្នុង AppState ជាសកលពេលជោគជ័យ)។
  • Forgot Password (ភ្លេចពាក្យសម្ងាត់): Pop-up dialog to request recovery link (ផ្ទាំងស្នើសុំតំណភ្ជាប់សារឡើងវិញ)។

🔌Call API Test / ការតេស្តភ្ជាប់ API

Network Flow
  • Unified Dynamic Fetching (ការទាញទិន្នន័យឆ្លាតវៃ): Requests JSON database from raw GitHub URL online (ទាញយកទិន្នន័យ JSON ពី GitHub តាមរយៈអ៊ីនធឺណិត)។
  • Robust Offline Fallback (ប្រព័ន្ធរក្សាទិន្នន័យបម្រុង): Reverts to local `db.json` on timeout/failure (ប្តូរទៅអានឯកសារក្នុងម៉ាស៊ីនពេលគ្មានអ៊ីនធឺណិត ឬយឺតខ្លាំង)។
  • External API Integration (បណ្ណាល័យឌីជីថល): Calls Open Library API to fetch computer science book listings (ទាញយកបញ្ជីសៀវភៅកុំព្យូទ័រថ្មីៗពី Open Library)។
  • Release APK Permissions Fix (ការដោះស្រាយបញ្ហាដំឡើង APK): Added INTERNET permission to avoid network image/data failures (បន្ថែមសិទ្ធិអ៊ីនធឺណិតដើម្បីបង្ហាញរូបភាព និងទិន្នន័យលើទូរស័ព្ទ)។
